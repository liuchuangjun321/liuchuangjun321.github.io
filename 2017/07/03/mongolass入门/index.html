Mongolass

Mongolass 保持了与 mongodb 一样的 api，又借鉴了许多 Mongoose 的优点，同时又保持了精简。所以可以直接参考mongodb的API即可，较大程度上简化了学习的内容，对于我这个渣渣来说也算是Mongolass的一大优点吧！
