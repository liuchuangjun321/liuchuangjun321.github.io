Q1：今天看Express代码时遇见一行代码app.use('/signup', require('./signup'));平时大多对这样的app.use(function(req, res, next) { … });使用中间件的较为习惯，所以看到这个比较懵逼

查了一下app.use()源码，解答的还算是比较详细的，来看官网的解释：

app.use([path,] callback [, callback...])，在指定的路径上挂载指定的中间件功能或函数：当所请求的路径的基础与路径匹配时，执行中间件功能。不仅如此，官网对于path和callback的类型还做了详细的罗列，感兴趣的请移步上述官网。

那也就是说require('./signup')返回的应该是一个function类型，回调函数嘛，希望这样理解没有错！
